## Featured Projects

### 🧠 Alzheimer's Disease Classification via Machine Learning
End-to-end ML pipeline for early classification of Alzheimer's disease using clinical and neuroimaging datasets.
- **Technologies:** Python, Scikit-learn, Pandas, NumPy, PCA
- **Highlights:** Compared Random Forest, SVM, Logistic Regression, AdaBoost, and KNN with stratified cross-validation; applied KNN imputation, normalization, and PCA for dimensionality reduction; achieved **97% accuracy**
- **🔗 [View Repository](https://github.com/micaeltoscano)**

### 📊 ENEM Performance & Absenteeism Classification (5M+ records)
Scalable ML models to classify student performance and predict absenteeism from large-scale educational data.
- **Technologies:** Python, Pandas, Scikit-learn, Keras, Parquet
- **Highlights:** Full data pipeline with feature engineering and cross-validation on 5M+ rows; CSV → Parquet conversion for I/O optimization; Decision Tree, Random Forest, SVM, and Neural Networks; **71% accuracy** on performance, **70%** on absenteeism
- **🔗 [View Repository](https://github.com/micaeltoscano/ENEM-Classificacao-Desempenho-Absentismo)**

### 🎵 Music & Mental Health Analysis
Exploratory and statistical analysis of the correlation between musical habits and mental health indicators.
- **Technologies:** Python, Pandas, Matplotlib, Seaborn
- **Highlights:** Original algorithm for quantifying lyrical lexical complexity; sentiment analysis; interactive visualizations
- **🔗 [View Repository](https://github.com/micaeltoscano/Saude_Mental_e_Musica)**

### 💰 Budget Data Analysis — Paraíba
Applied research on public budget growth and spending trends in the state of Paraíba.
- **Technologies:** Python, Pandas, Statistical Methods
- **Highlights:** Budget trend analysis, population correlation, pandemic impact study
- **🔗 [View Repository](https://github.com/micaeltoscano/Analise_Dados_Paraiba)**

### ✂️ Django CRUD — Barber Shop Management
Full-featured web application for barbershop management with complete CRUD functionality.
- **Technologies:** Django, Python, SQLite, HTML/CSS
- **Highlights:** User authentication, appointment and service management
