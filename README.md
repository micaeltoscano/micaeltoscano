I'm a Data Science & AI undergraduate at UFPB (Brazil) focused on turning
data into working systems — from large-scale pipelines to LLM-powered APIs.

My work spans predictive modeling, NLP, and retrieval-augmented generation (RAG),
with a habit of taking things end-to-end: raw public data → trained model → deployed API.
